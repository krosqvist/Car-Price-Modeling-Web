Description

This software estimates the current market value of any internal combustion engine car and projects its price development up to five years into the future. Powered by a random forest machine learning model, it has been trained on data from thousands of real used car sales across the UK. You can download the software from our GitHub repository and run it locally on your computer by following the provided setup instructions.
