Description

This software estimates the current market value of any internal combustion engine car and projects its price development up to five years into the future. The best estimates are made for average and common cars. It is not recommended to attempt estimating prices for rare or high-end cars as their pricing can not be generalized. The vehicles are assumed to be in a good condition and to include only the stock features. If these conditions are not met, the estimate can be inaccurate. You can download the software from our GitHub repository and run it locally on your computer by following the provided setup instructions.
